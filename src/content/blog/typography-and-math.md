学术写作经常要用到公式、脚注和表格。这篇演示它们在本博客里的样子。

## 行内与块级公式

行内公式用单个美元符号，例如质能方程 $E = mc^2$，或者高斯积分 $\int_{-\infty}^{\infty} e^{-x^2}\,dx = \sqrt{\pi}$。

块级公式用两个美元符号，会单独居中显示：

$$
\hat{f}(\xi) = \int_{-\infty}^{\infty} f(x)\, e^{-2\pi i x \xi}\, dx
$$

矩阵、求和这些也没问题[^katex]：

$$
\sum_{k=0}^{n} \binom{n}{k} x^k = (1 + x)^n
$$

## 脚注

学术写作常用脚注来补充说明或标注引用[^note]。在正文里写 `[^标记]`，在文末写对应的内容即可。

## 表格

| 方法 | 时间复杂度 | 空间复杂度 |
| ---- | ---------- | ---------- |
| 快速排序 | $O(n \log n)$ | $O(\log n)$ |
| 归并排序 | $O(n \log n)$ | $O(n)$ |
| 冒泡排序 | $O(n^2)$ | $O(1)$ |

## 代码

```python
def fib(n: int) -> int:
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

[^katex]: 公式由 KaTeX 渲染，速度快、无需联网加载。
[^note]: 脚注会自动编号，并在文末生成带返回链接的列表。
